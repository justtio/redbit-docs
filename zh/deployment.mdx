# 部署与本地引擎

Redbit 可以作为托管 Web 应用运行，也可以按需配对本地引擎以获得桌面侧能力。Web 应用负责 UI、模型配置和浏览器侧状态；本地引擎提供经过配对保护的本地媒体、自动化、插件或 MCP 操作。

## 托管 Web 应用

将 Vite 构建产物部署到你选择的静态托管平台。生产配置应将浏览器流量指向可信的 Supabase Functions 或你自己的兼容代理端点。

```bash
pnpm install
pnpm build
```

## 本地引擎

本地引擎监听 `127.0.0.1:19831`，受保护操作需要先完成配对。用户只应配对自己在本机启动的本地引擎。

## 安全注意事项

- 不要在前端环境变量中发布真实 API Key。
- 共享生产密钥应通过服务端函数或可信中继处理。
- 文件系统、自动化、插件和依赖安装等操作必须保持本地引擎配对保护。
- 将敏感内容发送给第三方模型前，请先确认供应商条款和数据处理要求。
