# 截图制作规范（内部参考）

> 本文件供截图制作参考，不面向用户。

## 总体要求

| 项目 | 要求 |
|------|------|
| 格式 | PNG（无压缩），2x 分辨率截图 |
| 尺寸 | 宽度 1200px（高度按比例） |
| 命名 | `guide-{序号}-{简短描述}.png`，例：`guide-1-1-volcano-register.png` |
| 标注颜色 | 红色（`#E53E3E` 或 `#EF4444`）用于箭头、方框、圆圈标注 |
| 标注线宽 | 箭头和方框线宽 3px，圆圈线宽 3px |
| 敏感信息处理 | API Key、邮箱地址等需打码（马赛克或纯色覆盖） |

## 标注规范

- **箭头**：红色实线箭头，指向被标注的目标元素，箭头末端不超过元素边缘
- **方框**：红色圆角矩形（圆角 8px），框住所指区域，不遮挡文字
- **圆圈**：红色正圆形，圈住按钮或小图标
- **编号**：如果一张图有多个标注点，在标注旁加白色圆形数字标签（① ② ③）
- 不在无关区域画线标注

## 敏感信息打码清单

截图前或截图后，以下内容必须打码处理：

- API Key（用黑色色块完全覆盖）

- 手机号（模糊或黑色覆盖）
- 身份证号（模糊或黑色覆盖）
- 真实姓名（模糊或黑色覆盖）
