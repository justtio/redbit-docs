# 透明的 BYOK 定价

Redbit 采用 **BYOK（Bring Your Own Key）** 模式。你自备 AI 模型的 API Key（Veo、Flux、Suno 等），我们提供生产级的 Studio 工作台。**模型调用零加价。**

<Info>
  订阅仅解锁**平台功能**。模型生成费用由你的 API Key 提供商（OpenAI、Google 等）直接收取。
</Info>

## 常见问题

<AccordionGroup>
  <Accordion title="我已经有 API Key 了，为什么还要订阅？" icon="sparkles">
    供应商控制台通常围绕单个端点设计。Redbit 提供这些端点之上的工作区层：卡片管理、参考控制、可视化素材管理、批量操作、创意工坊和可复用项目结构。
  </Accordion>
  <Accordion title="我的 API Key 安全吗？" icon="shield-check">
    Settings 会把敏感字段保存到浏览器存储，并在可用时先加密再写入 IndexedDB。供应商请求会按你配置的路由执行，例如直接供应商或中继。请把 key 放在 Settings，避免粘贴到 prompt 或聊天中。
  </Accordion>
  <Accordion title="可以取消或退款吗？" icon="rotate-left">
    所有方案均为一次性套餐（非订阅制），没有自动续费。退款资格请参阅我们的[退款政策](/refund)。
  </Accordion>
</AccordionGroup>
