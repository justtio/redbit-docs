# Redbit AI Studio

Redbit 是一个面向创作和自动化的工作台，可在同一个界面里管理图像、视频、音频、描述和多步任务。它由卡片画布、素材码头、模型配置和 Agent 编排能力组成。

与单一聊天线程不同，Redbit 让每一次生成或任务都可以作为独立卡片存在。你可以对比结果、分叉创意、复用素材，并保持相关工作在画布上可见。

## 数据与执行边界

Redbit 作为浏览器应用运行，也可以按需配对本地引擎。浏览器侧数据受用户浏览器配置影响；外部模型和集成只会在用户配置凭证并启动相应工作流时使用。

涉及敏感数据时，请先审查你的模型供应商设置、浏览器存储策略和本地引擎配对策略。
