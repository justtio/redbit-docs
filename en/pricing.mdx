# Transparent BYOK Pricing

Redbit operates on a **BYOK (Bring Your Own Key)** model. You provide your own API keys for AI models (Veo, Flux, Suno, etc.), and we provide the production-grade Studio workspace. **Zero markup on model calls.**

<Info>
  Your subscription unlocks **platform features only**. Model generation costs are charged directly by your API key provider (OpenAI, Google, etc.).
</Info>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Why subscribe if I already have my own API keys?" icon="sparkles">
    Provider consoles are usually optimized for individual endpoints. Redbit provides the workspace layer around those endpoints: card management, references, visual asset management, batch operations, Creative Workshop, and reusable project structure.
  </Accordion>
  <Accordion title="Is my API key safe?" icon="shield-check">
    Settings stores sensitive fields in browser storage and encrypts them before IndexedDB persistence when available. Provider requests follow the route you configure, such as direct provider or relay. Keep keys in Settings and avoid pasting them into prompts or chat.
  </Accordion>
  <Accordion title="Can I cancel or get a refund?" icon="rotate-left">
    All plans are one-time packages (not subscriptions). There are no auto-renewals. For refund eligibility, please see our [Refund Policy](/refund).
  </Accordion>
</AccordionGroup>
