# Redbit AI Studio

Redbit is a creative workspace for running image, video, audio, and automation workflows from one interface. It combines a card-based canvas, an asset dock, model configuration, and an agent layer that can orchestrate multi-step tasks.

The product is designed for creators and teams who want more control than a single chat thread provides. Each generation or task can live as an independent card, so you can compare results, branch ideas, reuse assets, and keep related work visible.

## Data and execution model

Redbit runs as a browser application and can optionally pair with a local engine. Browser-stored data remains under the user's browser profile, while external model providers and configured integrations are used only when the user supplies credentials and starts a workflow that requires them.

For sensitive deployments, review your own provider settings, browser storage policy, and local-engine pairing policy before production use.
